# 과제
## 1번 문제
두 자연수 A와 B를 입력 받아 A와 B의 합, A에서 B를 뺀 값, A와 B의 곱, A에서 B를 나눈 몫과 나머지를 출력하는 프로그램을 작성하세요. 
실행 예시
```
A >> 30
B >> 7
37
23
210
4
2
```

## 3번 문제

```
[3, 6, 9]
```
